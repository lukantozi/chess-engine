# python-chess

Command-line chess game written from scratch in Python, with a simple AI opponent based on minimax and alpha–beta pruning.

- Implements standard move rules, including castling, promotion, and basic check/checkmate handling
- Uses an object-oriented board and piece model
- Supports saving and resuming games from `saved_game.json`

## Run

```bash
python project.py
```

Enter moves as `e2e4`. Enter `q` to save and quit.
